# STM32G4 FOC 电机控制项目

基于 STM32G431 单片机的高性能 FOC（Field-Oriented Control）电机驱动系统。

## 📋 项目概述

本项目实现了完整的无刷直流电机（BLDC/PMSM）FOC控制方案，支持多种控制模式和角度获取方式。

## ✨ 功能特性

### 控制模式
| 模式 | 功能 | 说明 |
|------|------|------|
| **模式1** | 电压开环 | 直接给定 dq 轴电压指令 |
| **模式2** | 电流闭环 | 精确控制 dq 轴电流 |
| **模式3** | 速度闭环 | 恒定转速控制 |
| **模式4** | 位置闭环 | 精确位置控制 |
| **模式5** | 无感控制 | SMO 滑模观测器无传感器控制 |

### 角度获取方式
- **编码器反馈**：支持增量式编码器
- **开环角度自增**：用于电机启动阶段
- **SMO 无感估算**：滑模观测器估算转子位置

### 保护功能
- 过流保护（4A 阈值）
- 母线电压监测
- 自动零漂校准

### 通信接口
- **USART1**：串口参数配置、实时示波器数据传输
- **SPI1**：扩展接口

## 🛠️ 硬件需求

### 主控芯片
- **STM32G431RBT6**

### 外设接口
| 外设 | 用途 |
|------|------|
| **ADC** | 三相电流采样 + 母线电压采样 |
| **TIM1** | PWM 输出（SVPWM） |
| **TIM3** | 编码器计数 |
| **USART1** | 串口通信 |

### 电机参数（可配置）
- 极对数：7（默认）
- 编码器脉冲数：16384（4倍频后）
- 母线电压：12V
- 最大转速：3000 r/min
- 电流限制：2.0A

## 🚀 快速开始

### 1. 环境要求
- **Keil MDK5** 或更高版本
- **STM32CubeMX**（可选，用于配置外设）

### 2. 编译与烧录
1. 使用 Keil MDK 打开 `MDK-ARM/G431TEST.uvprojx`
2. 配置编译选项（Target: STM32G431RBT6）
3. 编译工程
4. 使用 ST-Link 烧录到开发板

### 3. 串口配置
- 波特率：115200
- 数据位：8
- 校验位：None
- 停止位：1

## 🔧 参数配置

### 主要配置参数（MotorConfig.h）

```c
#define Pn              7           // 电机极对数
#define PluseNum        16384       // 编码器每转脉冲数
#define DCBUSVALUE      12.0f       // 母线电压(V)
#define SPEED_LIM       3000        // 最大转速(r/min)
#define CURRENT_LIM     2.0f        // 电流限制(A)
#define VOLTAGE_LIM     6.0f        // 电压限制(V)

// PI参数
#define IDKP            1.5f        // d轴电流环Kp
#define IDKI            0.05f       // d轴电流环Ki
#define IQKP            1.5f        // q轴电流环Kp
#define IQKI            0.05f       // q轴电流环Ki
#define SPEED_KP        0.5f        // 速度环Kp
#define SPEED_KI        0.02f       // 速度环Ki
```

## 📡 通信协议

### 参数读写
支持通过串口动态修改参数：
- 16位/32位整数参数读写
- 浮点型参数读写

### 示波器功能
实时传输4路信号到上位机显示：
- Uref: 参考电压
- Iact: 实际电流
- Speedact: 实际转速
- Motor_Bit16: 电机状态位
