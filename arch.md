# 项目架构文档

## 1. 项目概述
本项目为开发板配套软件，主要用于电机控制（FOC算法）及相关功能实现。

## 2. 目录结构
- **`.mxproject`**: 项目配置文件。
- **`G431TEST.ioc`**: 开发板配置文件。
- **`Drivers`**: 驱动相关代码或库。
- **`Foc`**: 电机控制（FOC算法）实现。
- **`Function`**: 功能模块代码。
- **`Inc`**: 头文件目录。
- **`MDK-ARM`**: Keil MDK-ARM 工程文件。
- **`Src`**: 源代码目录。

## 3. 模块划分
1. **驱动层（Drivers）**: 提供硬件驱动支持。
2. **控制层（Foc）**: 实现电机控制算法。
3. **功能层（Function）**: 实现具体功能模块。
4. **应用层（Src）**: 主程序及业务逻辑。

## 4. 依赖关系
- 驱动层为控制层提供硬件支持。
- 控制层依赖驱动层实现电机控制。
- 功能层依赖控制层和驱动层。
- 应用层整合所有模块实现完整功能。
